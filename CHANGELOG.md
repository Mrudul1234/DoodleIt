# Changelog

## [1.0.0]

### Added
- Miro-style infinite whiteboard with object-based model
- Pen tool with Catmull-Rom smooth interpolation
- Arrow / line tool with arrowhead
- Eraser that removes pen strokes on hover
- Sticky notes in 8 colors, draggable and resizable
- 8 shape types: rect, circle, triangle, diamond, star, hexagon, arrow, speech bubble
- Text tool with Caveat handwriting font
- Image upload and drag-and-drop
- 12 preset colors + custom color picker
- Stroke size (1–40px) and opacity (5–100%) controls
- Unlimited undo/redo
- Dark and Light theme toggle (clean flat dark mode, no glow effects)
- Export board as PNG (VS Code webview compatible)
- Zoom with scroll wheel, +/− buttons, Fit button
- Right-click context menu: Duplicate, Copy, Paste, Bring to front, Send to back, Delete
- Full keyboard shortcuts
- Ctrl+Shift+D to open
